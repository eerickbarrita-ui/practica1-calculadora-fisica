Descripción del Proyecto

Este proyecto consiste en una calculadora de cinemática desarrollada en C#. La aplicación permite calcular velocidad, distancia y tiempo a partir de los datos proporcionados por el usuario.

Durante esta práctica se refactorizó el código original para implementar los modificadores ref y out, comprendiendo cómo funcionan los parámetros por referencia y cómo permiten modificar o devolver valores sin utilizar retornos explícitos.

⸻

Funcionalidades

* Calcular velocidad (m/s).
* Calcular distancia (m).
* Calcular tiempo (s).
* Validación de entrada de datos.
* Implementación de parámetros por referencia mediante ref.
* Implementación de parámetros de salida mediante out.

⸻

Instrucciones para Compilar y Ejecutar

1. Abrir una terminal en la carpeta del proyecto.
2. Restaurar dependencias (si es necesario):

dotnet restore

3. Compilar el proyecto:

dotnet build

4. Ejecutar la aplicación:

dotnet run

Reflexión Personal

Durante esta práctica aprendí la diferencia entre pasar parámetros por valor y por referencia en C#. Comprendí que el modificador ref permite trabajar directamente sobre una variable existente, mientras que out se utiliza para devolver resultados desde un método sin necesidad de usar return. También reforcé mis conocimientos sobre el uso de Git y GitHub mediante la creación de ramas, commits y la integración de cambios al proyecto principal. Esta actividad me ayudó a entender mejor cómo se gestionan los datos en memoria y la importancia de mantener un flujo de trabajo ordenado durante el desarrollo de software.
